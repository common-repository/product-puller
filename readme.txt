=== You can pull product data from Amazon without affiliate api ===

Product Puller is a plugin that helps you pull product data from the internet. Currently able to fetch data from Amazon. You do not need affiliate api for use this plugin.

=== Features ===

* Allows you to add your Associate ID to links.
* You can pull the data from all amazon sites.
* You can fetch the product data from Amazon to WooCommerce (with Subscribe option)
* You can create custom product comparisons or pull the ready one from Amazon. (with Subscribe option)
* You can add the product details to your posts in many ways.
* You can translate the plugin according to your language.
